# How-To Guides

Task-oriented guides for accomplishing specific goals with the Modern Django API Template.

## Available Guides

| Guide | Description |
|-------|-------------|
| [Add a New Domain](add-new-domain.md) | Complete checklist for adding a new business domain (models, services, controllers) |
| [Custom Exception Handling](custom-exception-handling.md) | Map domain exceptions to HTTP status codes in controllers |
| [Override IoC in Tests](override-ioc-in-tests.md) | Mock services and dependencies in integration tests |
| [Add a Celery Task](add-celery-task.md) | Quick reference for creating background tasks |
| [Secure Endpoints](secure-endpoints.md) | Add authentication and rate limiting to API endpoints |
| [Configure Observability](configure-observability.md) | Set up Logfire or other OpenTelemetry backends |

## When to Use These Guides

These guides assume you understand the basic concepts of the template. If you are new to the project, start with the [Getting Started](../getting-started/index.md) section and [Tutorial](../tutorial/index.md) first.

Each guide focuses on a single task and provides step-by-step instructions with code examples you can adapt to your needs.
